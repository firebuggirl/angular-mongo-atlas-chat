# Server Side Rendering for SEO

  -  server side rendering with Angular Universal

  -  try in a fresh project created with

  `ng new <my-project-name>`

  ` ng add @nguniversal/express-engine --clientProject <my-project-name>
  `

  ` npm run build:ssr
  `

  ` npm run serve:ssr
  `
